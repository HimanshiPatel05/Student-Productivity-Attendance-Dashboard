# Student Productivity & Attendance Analytics Dashboard

## Project Overview

This Power BI dashboard analyzes student performance using attendance, GPA, study hours, and academic metrics.

## Tools Used

- Power BI
- Excel

## KPIs

- Total Students
- Average Attendance
- Average GPA
- Average Study Hours

## Dashboard Pages

### 1. Overview Dashboard

Contains:
- KPI Cards
- Attendance vs GPA Scatter Plot
- Branch-wise GPA Analysis
- Gender Distribution

### 2. Risk Analysis Dashboard

Contains:
- High Risk Students
- Average GPA of High Risk Students
- Branch-wise Risk Analysis
- Gender-wise Risk Analysis

## Dataset

250 Student Records
